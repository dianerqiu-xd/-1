# Vibe Coding 图像颜色空间、图像插值作业小结

## 一、实验目标

本次实验使用 Python、OpenCV、NumPy 完成图像颜色空间与图像插值处理。程序可以读取输入图像，输出 RGB、HSV、YCrCb、Lab 等颜色空间的各通道图像，并实现最近邻插值与双线性插值，支持图像放大、缩小、旋转拉伸等操作。

## 二、实现内容

1. 颜色空间部分
   - 输入图像由 OpenCV 读取后从 BGR 转换为 RGB。
   - RGB 通道直接拆分为 R、G、B 三个灰度通道，并额外保存单色可视化图。
   - HSV 使用手写 `rgb_to_hsv_manual` 函数实现，按照最大值、最小值和差值计算 H、S、V。
   - 扩展输出 YCrCb、Lab 两类颜色空间通道，便于观察亮度、色度和感知颜色分量的差异。

2. 插值部分
   - 最近邻插值：目标像素反向映射到源图像坐标后，选择距离最近的源像素，速度快但放大时容易出现块状感。
   - 双线性插值：目标像素反向映射后，取源图像相邻四个像素，根据横向和纵向距离加权平均，结果更平滑。
   - 支持放大、缩小、旋转拉伸，并分别输出最近邻与双线性结果。

3. 测试截图
   - `outputs/screenshots/color_space_channels.png` 展示颜色空间通道拆分结果。
   - `outputs/screenshots/interpolation_results.png` 展示放大、缩小、旋转拉伸结果。
   - `outputs/screenshots/interpolation_detail_compare.png` 对比最近邻和双线性插值的局部细节。

## 三、核心算法说明

HSV 转换中，先将 RGB 归一化到 0 到 1，根据三个通道最大值和最小值计算色相、饱和度和明度。色相由最大通道决定：R 最大、G 最大、B 最大时分别使用不同公式，并映射到 OpenCV 常用的 0 到 179 范围。

插值部分统一采用反向映射思想：遍历输出图像每个像素，计算它在输入图像中的对应坐标。最近邻直接取整得到源像素；双线性插值根据小数坐标周围四个像素进行两次线性加权。

## 四、实验结果观察

RGB 通道能够直观看出原图在红、绿、蓝三个分量上的强弱。HSV 的 H 通道强调色相变化，S 通道反映颜色纯度，V 通道接近图像明暗信息。YCrCb 和 Lab 通道进一步把亮度和色彩信息分离，适合后续图像增强、分割或颜色分析。

插值结果中，最近邻插值边缘更硬，局部放大会产生明显台阶；双线性插值边缘更平滑，视觉效果更自然。旋转拉伸时，双线性插值在斜边和纹理区域的锯齿更少，但计算量高于最近邻。
